# GDHive - v1.0

Ein modulares Terminal-Dashboard für System-Management und Automatisierung, entwickelt von **GreenDataDev**.

## Features
* **System-Status**: Schneller Überblick über CPU, RAM und Speicher.
* **Datei-Organizer**: Automatische Sortierung von Dateien in Kategorien (Bilder, Dokumente, Programmierung etc.).
* **Hardware-Spezifikationen**: Detaillierte Auslese von Hardware-Komponenten, Netzwerk-Schnittstellen und Laufwerken.

## Installation & Start
1. Repository klonen: `git clone https://github.com/GreenDataDevAndroid/GDHive.git`
2. Abhängigkeiten installieren: `pip install psutil`
3. Starten: `python main.py`
